# AES-256

Este repositório contém um exemplo prático de criptografia e descriptografia usando o algoritmo **AES-256** no modo **GCM**, com **padding PKCS#7**, desenvolvido em **Java**.

## Funcionalidades
- Geração de chave AES-256 (32 bytes)
- Aplicação de padding PKCS#7 se necessário
- Criptografia AES/GCM/NoPadding
- Exibição de IV e ciphertext em Base64
- Descriptografia e validação do texto original

## Tecnologias usadas
- Java (javax.crypto)
- IntelliJ IDEA
